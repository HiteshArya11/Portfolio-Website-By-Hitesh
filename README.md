# 🚀 Run Locally

Because the project uses Flask, you must run it through Python instead of opening files directly.

---

## Step 1 — Open Terminal

Open CMD / PowerShell / Terminal inside the project folder.

---

## Step 2 — Navigate to Project Folder

```bash
cd portfolio
```

---

## Step 3 — Install Flask

```bash
pip install flask
```

---

## Step 4 — Run Application

```bash
python app.py
```

---

## Step 5 — Open Browser

Visit:

```text
http://127.0.0.1:5000
```

🎉 Your portfolio website should now be running locally.
